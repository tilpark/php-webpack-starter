# Php Webpack Starter

## Setup instructions

### Download
```bash
git clone https://github.com/tilpark/php-webpack-starter.git
cd php-webpack-starter
```
### Install dependencies
#### Composer
```bash
composer install
```
#### Webpack
```bash
yarn
```
### Usage
##### Start Webpack Development Server
```bash
yarn start
```
##### Build Webpack
```bash
yarn build
```
